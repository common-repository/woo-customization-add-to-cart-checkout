== Description ==
Woo Customization use to change the add to cart button text and check out button text. Use for very type of woocommerce product cart button text with indvedual like simple product cart button text.
veriable cart button text, group cart button text,


== Features ==
* Change the Deffulat Cart button text.
* Change Cart button text on Shop Page.
* Change the Cart button text on Single page.
* Change the Every type of Product cart button text on single product page.
* Change the Every Type of Product Cart Button text on Shop Page.
* Just Copy the ShortCode and use it any where. 
* Easy to Use.
* Saves all changelog in cache to keep things optimized

== Installation ==
To add a WordPress Plugin using the built-in plugin installer:

Go to Plugins > Add New.

1. Type in the name \"WP Change Log Scraper\" in Search Plugins box
2. Find the \"WP Change Log Scraper\" Plugin you wish to install.
3. Click Install Now to install the WordPress Plugin.
4. The resulting installation screen will list the installation as successful or note any problems during the install.
If successful, click Activate Plugin to activate it, or return to plugin installer for further actions.
